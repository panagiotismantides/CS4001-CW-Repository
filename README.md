Gadget Shop Management System

Java coursework project using BlueJ and Swing GUI.

Features:

- Add Mobile phones
- Add MP3 players
- Make phone calls
- Download music
- Display all gadgets

Classes:

- Gadget (base class)
- Mobile (extends Gadget)
- MP3 (extends Gadget)
- GadgetShop (GUI controller)
